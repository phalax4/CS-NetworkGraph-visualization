Gouda CS 337 Fall 2008 Quiz 1 ---------------------------------------------------------------------------- Consider the following codes: Code 1: --------------------------------- codeword | occurrence probability --------------------------------- 0 | .1 1 | .7 10 | .2 --------------------------------- Code 2: --------------------------------- codeword | occurrence probability --------------------------------- 0 | .6 10 | .1 11 | .3 --------------------------------- Code 3: --------------------------------- codeword | occurrence probability --------------------------------- 1 | .1 01 | .6 00 | .2 --------------------------------- Which of these codes is a prefix code and which of them is not? Explain your answers. ------------------------------------------------------------------------------ Student Name: Student EID: ------------------------------------------------------------------------------ Solution: Code 1 is not a prefix code because codeword "1" is a prefix of codeword "10". Code 3 is not a prefix code either because the occurrence probability does not add up to 1 (so it is not even a code). Only code 2 is a prefix code because the occurrence probability adds up to 1 and no codeword is a prefix of any other codeword.

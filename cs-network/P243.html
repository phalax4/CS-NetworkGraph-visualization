Try Some Lisp

It is easy to try a little Lisp on CS Unix machines:

  • To start, enter lisp or gcl
  • To exit, enter (bye)
  • If you get an error, enter :q

Some things to Try:

(+ 2 3)

(sqrt 2)

(cons 'a '() )

(cons 'a '(b) )

(cons 'a (list 'b 'c) )

(defun twice (x) (* 2 x))

(twice 3)

(defun fact (n) (if (<= n 1) 1 (* n (fact (- n 1)))))

(fact 5)

(fact 100)

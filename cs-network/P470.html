Gouda CS 337 Fall 2007 Quiz 6 ------------------------------------------------------------------------------- Design a Finite State Machine M over the alphabet {a, b, c, .., z} such that M accepts a string iff the string has exactly four x's. ----------------------------------------------------------------------------- Student Name: EID: Student Section: ----------------------------------------------------------------------------- Solution: Let A be the alphabet {a, b, c, ..., z} and let A' be A - {x}. The finite state machine over alphabet A has six states: 1, 2, 3, 4, 5, 6. State 1 is the initial state and state 5 is the only accepting state. Each state i, where i = 1 to 5, has two outgoing transitions: a self-loop labeled A' and a transition labeled {x} to state i+1. State 6 has only one outgoing transition, a self-loop labeled A.

Gouda CS 337 Fall 2007 Quiz 10, 11 ------------------------------------------------------------------------------- Define a Haskell function cumsum that takes a non-empty list xs of integers and computes a list of the cumulative sums of the elements in xs. For example, if xs is the list [x, y, z, ..], then cumsum computes the list [x, x+y, x+y+z, ..]. ----------------------------------------------------------------------------- Student Name: EID: Student Section: ----------------------------------------------------------------------------- Solution: listadd x [y] = [x+y] listadd x (y:ys) = (x+y):(listadd x ys) cumsum [x] = [x] cumsum (x:xs) = x:(listadd x (cumsum xs))

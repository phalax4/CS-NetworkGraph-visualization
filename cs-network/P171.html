Electronic Communication for CS354R

Using Piazza

We will use Piazza for class discussion, so that other students, as well as the TA and myself, can participate in answering questions. We will also use Piazza for any important or last minute announcements, so not joining could have ramifications your final grade. If you have any problems or feedback for the developers, email team@piazza.com.

Using Canvas

In addition to information posted on the class webpage and in Piazza you will be able to find your course grades as the semester progresses via Canvas.

Note that we will not generally make information that is available on the class webpage or in Piazza available on Canvas as well, you should rely on it primarily for grades.

Course Description:

This course is an introduction to programming in C++. You should already know the basics of programming, this course will focus on learning c++ itself (which is huge and we have a short time). For a 1-credit course this will be intensive, ~7 quizzes and 7 assignments.

Syllabus

The course will cover most of the textbook. The course will start by going over basic syntax, types, and control flow. I/O operations are next followed by arrays, pointers, and dynamic memory management. Afterwards the c++ containers will be taught followed by structures and classes-- a large area that will include operator overloading, inheritance, and polymorphism. The class will finish with a larger project to tie the concepts together.

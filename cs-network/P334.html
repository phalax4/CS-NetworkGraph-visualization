Electronic Communication for CS354

Using Piazza

This semester we will be using Piazza for class discussion. The system is highly catered to getting you help fast and efficiently from classmates, the TA, and myself. Rather than emailing questions directly to me or the TA, I encourage you to post your questions on Piazza. We'll get them just as fast, and in addition other students can participate in the discussion. If you have any problems or feedback for the developers, email team@piazza.com.

While Piazza is great for discussions, it also provides for information like the syllabus, assignments, etc. that is found on the course webpage. We will not in general replicate this information both places. In the past, things other than discussions have been provided via the webpage, and we plan to continue that this semester, although we may start migrating more material to Piazza if it proves convenient.

Most of you will have been automatically enrolled for this class in Piazza, but if you're not, you can enroll yourself. Once enrolled, the course page is here.

Using Canvas

In addition to information posted on the class webpage and in Piazza you will be able to find your course grades as the semester progresses via Canvas. If you are not familiar with Canvas, instructions for using it are available via the login page here. Note that we will not generally make information that is available on the class webpage or in Piazza available on Canvas as well, you should rely on it primarily for grades.

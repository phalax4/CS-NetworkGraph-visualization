Extra UML MetaModeling Problems


There are all sorts of graphs used in Computer Science.  Three are Pie Charts, Excel Line Charts, and Flow Charts.

1. Pie Charts

Below are two examples of pie charts.  Both are object diagrams.  The left is a 3D chart; the right is a 2D chart.
  1. What is the class diagram metamodel of Pie Charts that has the above charts as instances?
  2. What is a database schema that is derived from your class diagram?
  3. Populate a database with tuples that encodes the left pie chart.
  4. Populate another database with tuples that encodes the right pie chart.

2. Excel Line Charts

Below are two examples of Excel Line charts.  Both are object diagrams.
  1. What is the class diagram metamodel of Excel Line Charts that has the above charts as instances?
  2. What is a database schema that is derived from your class diagram?
  3. How many tuples are in each table for the left-Line chart?
  4. How many tuples are in each table for the right-Line chart?
